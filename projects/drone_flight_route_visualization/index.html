Flight Delay Predictions

Predicting airline delays using historical flight and weather data.

Overview

This project builds machine learning models to predict whether a flight will be delayed, leveraging airline metadata, temporal features, and weather conditions.

Tech Stack

  • Python
  • Pandas / NumPy
  • Scikit-learn
  • XGBoost
  • Databricks

Key Takeaways

  • Weather and departure time are the strongest predictors of delay
  • Tree-based models significantly outperform linear baselines
  • Feature engineering had a larger impact than model choice
