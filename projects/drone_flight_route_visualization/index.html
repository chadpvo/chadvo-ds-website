Tech Stack: HTML, D3.js, Plotly, JavaScript, Mapbox, GeoJSON, Python,Flask, Docker

Summary: An interactive web app using D3.js to map and analyze drone flight paths with elevation profiles and geospatial data overlays.

TL;DR: We built an interactive drone flight route visualizer, intended for drone operators and drone enthusiasts used to detect drone activity and models.

Feedback Loops For the Win

Besides learning the new tech stack, the biggest takeaway from this project was the importance of feedback loops through several usability case studies in machine learning systems. By continuously monitoring model performance and incorporating user feedback, we can ensure that our UIs remain accurate and relevant over time. This iterative approach not only enhances model reliability but also builds trust with end-users, ultimately leading to better decision-making and improved operational efficiency.

Note: This project was completed as part of our Data Visualization's Summer 2025 course. Therefore, technical details, especially on drone route analysis, may be limited as they were not weighted as much for grading purposes.

The project was initially hosted on Digital Ocean but due to limited credit, the hosting ended in late 2025. For full technical details, please check our Github Repository
